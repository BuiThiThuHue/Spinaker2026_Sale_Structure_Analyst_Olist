# Olist E-commerce Data Analysis Project

This project focuses on exploring and analyzing consumer behavior data (EDA) and building the revenue structure of the Olist e-commerce platform (Brazil).
The main objective is to clean the data, visualize core business metrics, and thereby provide in-depth analytical insights into customer segments, shipping costs, and delivery performance.

---

## 📌 Data Source
The data used in this project is the **Brazilian E-Commerce Public Dataset by Olist**, collected from actual orders in Brazil between 2016 and 2018.
* **Data Source:** [Kaggle - Brazilian E-Commerce Public Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
* **Description:** The dataset includes multiple anonymized tables linked together, including: order status, prices, payment performance, freight costs, customer geographic location, and user review scores.
